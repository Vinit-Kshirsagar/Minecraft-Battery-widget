# Minecraft Battery Widget

A Flutter app with a custom Android 4x1 home screen widget displaying battery level using Minecraft-style hearts.

## Features

- Full, half, and empty hearts based on battery percentage
- Color-coded hearts: green/orange/red
- Updates automatically when battery changes
- Flutter app integration

## Setup

1. Clone the repository
2. Run `flutter pub get`
3. Run on Android device: `flutter run`
4. Add the 4x1 home screen widget
